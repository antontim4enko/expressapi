Welcome to the Notes API v1.1

List of available requests :

Getting All notes

GET
https://reactnotesapi.herokuapp.com/api/getData

Add new note

POST
https://reactnotesapi.herokuapp.com/api/putData

Params:

  • ID - id of the note
  • Text - text of the note

Update note

PUT
https://reactnotesapi.herokuapp.com/api/updateNote

Params:

  • ID - id of the note
  • Text - text of the note

Deleting note

DELETE
https://reactnotesapi.herokuapp.com/api/note/:note-id

Params:

  • note-id - id of the note
